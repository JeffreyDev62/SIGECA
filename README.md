Este sistema fue hecho para una escuela primaria, con el objetivo de facilitar 
la gestion de calificaciones (Boletas) y automatizar procesos administrativos.
Entre los archivos del sistema encontrará un manual de instalación, y un archivo
"Leeme.txt", dicho archivo contiene las credenciales de usuario para acceder al
sistema...
Este Sistema fue creado con fines academicos. Personalmente considerado
como buen punto de partida en el mundo del software.
